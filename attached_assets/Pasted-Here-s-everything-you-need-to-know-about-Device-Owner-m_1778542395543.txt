Here's everything you need to know about Device Owner mode for PlainApp:

What is Device Owner?
Android has two privilege levels above normal apps:

Device Admin — can lock screen, set password policy. Revocable from Settings.
Device Owner — the highest privilege a non-system app can hold. One per device. Much harder to remove.
PlainApp already has the PlainDeviceAdminReceiver which supports both.

Requirements before running the command
The app must NOT be installed as a normal user app with accounts set up. Specifically:

No Google accounts (or any accounts) should be on the device when you run the command — OR the device must be freshly factory reset. This is Android's restriction to prevent enterprise abuse.
USB Debugging must be enabled — Settings → Developer Options → USB Debugging ON.
ADB must be authorized — plug in USB, accept the "Allow USB Debugging" prompt on the device.
PlainApp must already be installed on the device before running the command.
The easiest clean path:

Factory reset the device (or use a fresh device)
Skip Google account setup during first-boot wizard
Install PlainApp via ADB: adb install app-debug.apk
Run the Device Owner command
Then add your Google account afterward — that's fine
If you already have a Google account on the device, you have two options:

Remove all accounts → run command → re-add accounts (accounts added after are fine)
Or accept that you cannot set Device Owner without removing accounts first
The exact steps
# Step 1 — Verify ADB sees the device
adb devices
# Should show: <serial>    device
# Step 2 — (If needed) Remove all accounts from device first
# Settings → Accounts → remove each one
# Step 3 — Set Device Owner
adb shell dpm set-device-owner com.ismartcoding.plain/.receivers.PlainDeviceAdminReceiver
# Success output:
# "Active admin set to component {com.ismartcoding.plain/com.ismartcoding.plain.receivers.PlainDeviceAdminReceiver}
#  and it's the device owner."
How to REMOVE Device Owner (if you ever need to)
# Via ADB (before any accounts or if you still have ADB access)
adb shell dpm remove-active-admin com.ismartcoding.plain/.receivers.PlainDeviceAdminReceiver
# OR from the app itself — PlainApp can call dpm.clearDeviceOwnerApp() 
# (we can add a button for this in Settings)
# OR: Factory reset the device — always works
You cannot remove Device Owner from Settings → Security → Device Admin Apps the normal way once it's set as owner (that's why the PIN guard we built blocks that page).
